## Start (students)
1. Green **Code** button > **Codespaces > Create codespace on main**.
2. Wait for the container to build (first build takes a few minutes).
3. Pick one AI assistant from the sidebar and sign in:
   - **Claude Code**: sign in with a Claude Pro/Max account, or set an
     `ANTHROPIC_API_KEY` Codespaces secret.
   - **GitHub Copilot**: sign in with your GitHub account. You need
     Copilot access; verified students get Copilot Pro free via
     GitHub Education.
   - **OpenAI Codex**: sign in with a ChatGPT account, or set an
     `OPENAI_API_KEY` Codespaces secret.

## What is preinstalled
- R with tidyverse, mgcv (GAMs), forecast + fable/feasts (ARIMA),
  scoringutils v2, and hubverse tooling
- Quarto (for evaluation dashboards) and RStudio Server on port 8787
- Three AI assistants ready to sign in: Claude Code, GitHub Copilot,
  and OpenAI Codex (with the `codex` terminal CLI also installed)
